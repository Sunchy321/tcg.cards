# 隐私政策

最后更新：2026-04-19

本隐私政策说明 TCG Cards 如何处理与本仓库、已部署网站、面向 App 的接口、公开 API、数据库导出和相关工具有关的信息。

本文档是项目模板和运营政策，不构成法律意见。在公开上线前，应确认项目运营主体名称、联系方式、托管地区、分析工具和特定司法辖区的合规义务。

## 适用范围

本政策适用于 TCG Cards 项目运营的服务，包括：

- 支持的 TCG 公共网站
- 账号、套牌、收藏和设置功能
- 公开 API 和面向 App 的接口
- 数据库导出端点
- 授权运营者使用的内部控制台功能
- 项目运营的 watcher 和通知服务

本政策不适用于 TCG Cards 链接或集成的第三方网站、API、数据集或服务。这些第三方服务受其自身政策约束。

## 我们收集的信息

根据启用的功能不同，服务可能收集以下信息。

### 账号信息

- 邮箱地址
- 用户名或显示名称
- 认证系统处理的密码认证元数据
- 会话标识和登录状态
- 为账号或开发者访问创建的 API key
- 账号角色、权限和管理员状态

### 用户内容和偏好

- 套牌列表和相关套牌元数据
- 点赞、收藏、保存的卡牌或其他保存的用户操作
- 搜索设置、语言偏好和界面偏好
- 提交给助手或聊天功能的消息
- 反馈、issue 报告、下架请求或支持消息

### 技术和使用信息

- IP 地址
- 请求 URL、方法、状态码和时间戳
- 浏览器、设备和操作系统信息
- 来源页面和 user-agent 字符串
- API 使用情况、限流事件和错误日志
- 安全、滥用防护和运营日志

### 数据源和导入元数据

服务可能存储来源元数据，例如外部数据集名称、版本、hash、获取时间、上游 URL 和导入状态。这类信息通常不会识别具体用户。

## 我们如何使用信息

我们使用收集的信息来：

- 提供、运营和保护服务
- 认证用户并管理会话
- 支持账号、套牌、收藏、API key 和设置功能
- 响应 API 请求并提供数据库导出
- 监控服务健康状态、诊断 bug 和防止滥用
- 在配置后发送运营通知
- 回应支持、法律、更正或下架请求
- 改进搜索、数据导入和卡牌展示功能
- 履行法律义务并执行服务条款

## Cookie 和本地存储

服务可能使用 Cookie、本地存储或类似技术来实现：

- 认证会话
- 语言和展示偏好
- 安全和滥用防护
- 记住本地界面状态

如果未来加入分析、广告或追踪工具，应在启用这些工具前更新本政策。

## AI 和助手功能

如果启用助手或聊天功能，用户消息可能会发送到配置的 AI 提供商或兼容 API 端点以生成回复。服务也可能向该提供商发送相关卡牌、规则或搜索上下文。

请不要提交敏感个人信息、机密数据、支付信息、密钥、私钥，或任何你不希望外部提供商处理的信息。

AI 提供商会按照其自身条款和隐私政策处理提交的数据。项目运营方应在公开上线前记录当前使用的提供商。

## 我们如何共享信息

我们不会出售个人信息。

我们可能与以下对象共享信息：

- 托管和基础设施提供商，包括 Cloudflare Workers、R2、KV、Hyperdrive 及相关日志
- 数据库、存储和备份提供商
- 认证、邮件、通知和安全工具
- 使用助手功能时的 AI 提供商
- 用户提交 issue 或法律请求时涉及的源码托管和 issue 跟踪平台
- 在法律、合规、安全或滥用防护需要时的相关接收方

如果法律、法院命令、法律程序要求，或为了保护用户、项目或第三方的权利、安全和保障，我们也可能披露信息。

## 公开内容

如果相关功能启用，部分用户操作可能是公开的或对其他用户可见，例如公开套牌、用户名、评论或分享内容。请不要发布你不希望公开的信息。

## 数据保留

我们仅在本政策所述目的合理需要的期限内保留个人信息，包括运营、安全、法律合规、备份和争议处理。

公开上线前应配置典型保留规则：

- 账号数据会在账号保持活跃期间保留
- 会话和 API key 会保留到过期、撤销或账号删除为止
- 日志会在有限的运营和安全周期内保留
- 备份可能会在有限的备份生命周期内保留已删除数据
- 法律、滥用或下架记录可能会根据处理请求的需要保留

## 安全

我们使用合理的技术和组织措施保护信息，包括访问控制、环境密钥、数据库访问控制和基础设施安全能力。

没有任何线上服务能保证绝对安全。用户需要自行保护账号凭据和 API key 的安全。

## 你的选择和权利

根据你的所在地和适用法律，你可能拥有以下权利：

- 访问与你账号相关的个人信息
- 更正不准确的信息
- 删除账号或部分个人信息
- 反对或限制特定处理
- 请求获取部分信息的副本
- 在未来引入特定共享或营销用途时选择退出

如需提出请求，请创建 GitHub issue，并在标题中包含 `Privacy`，或使用项目运营方公布的联系方式。

我们可能需要验证你的身份后才能处理请求。出于安全、法律合规、备份、滥用防护或正当运营需要，部分信息可能会继续保留。

## 儿童

本服务不面向 13 岁以下儿童。如果你未满 13 岁，请不要使用本服务或创建账号。如果项目运营方发现收集了 13 岁以下儿童的个人信息，应在法律要求的情况下删除相关信息。

某些司法辖区可能适用额外年龄要求。

## 跨境处理

服务可能使用多个地区的基础设施运营。使用本服务即表示信息可能会在你所在国家或地区以外处理和存储，但仍受适用法律和提供商配置约束。

## 第三方链接和内容

服务可能链接到第三方网站、API、数据集、卡牌数据库、游戏发行方、图片托管或社区资源。我们不对这些第三方的隐私实践负责。

第三方游戏内容、商标、卡图、卡牌文本和源数据另见 [LEGAL.zh-CN.md](./LEGAL.zh-CN.md)。

## 政策变更

本政策可能随项目演进而更新。重大变更应记录在仓库中；对于公开服务，应在变更生效前或生效时以合理方式告知。

## 联系方式

如有隐私问题或请求，请创建 GitHub issue，并在标题中包含 `Privacy`，除非项目运营方公布了专门联系方式。
